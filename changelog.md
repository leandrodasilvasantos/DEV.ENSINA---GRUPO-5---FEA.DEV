# Changelog - Dev.ensina 

## Info
This changelog represents the project change events during the project development time for Fea.dev, the initial program of Dev.ensina.

- 23/09/2025
- Reunião:
- Foi discutido a fundamentação teórica do artigo "Risk-Averse Action Selection Using Extreme Value
Theory Estimates of the CVaR"
- A importância da fundamentação estatística foi perceber que não podemos estimar realmente o CVAR em métodos simples, onde há poucos pontos para analisar na cauda da distribuição
- Estimar melhor o CVAR nos ajudaria a aplicar em estratégias quantitativas
